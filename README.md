## Installation

Check if npm is installed in your environment:

```bash
npm -v
```

Install all project dependencies:

```bash
npm i
```

<br/>

## Setup

First, run the development server:

```bash
npm run dev
```

<br/>

## Build

Build the project when you want to access the .css files

```bash
npm run build
```
